# 📊 Full Stack Sales Analytics Dashboard

A full-stack business analytics platform built using the MERN stack.  
This application helps businesses manage products, monitor inventory, analyze revenue trends, and export business data for reporting.

---

## 🚀 Project Duration

**Feb 2025 – Apr 2025**

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- Chart.js
- React CSV
- HTML5, CSS3, JavaScript

### Backend
- Node.js
- Express.js
- REST APIs

### Database
- MongoDB
- Mongoose

### Tools & Deployment
- Git & GitHub
- Postman (API Testing)
- VS Code

## ⚙ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/mern-sales-analytics.git
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔐 Environment Variables

Create `.env` file inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```
